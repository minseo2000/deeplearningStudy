# 텍스트 특징 추출 하기

- 모듈 목록

| 모듈 이름             | 설명                                                              |
|-------------------|-----------------------------------------------------------------|
| CountVectorizer   | 단순히 각 텍스트에서 횟수를 기준으로 특징을 추출하는 방법이다.                             |
| TfidVectorizer    | TF-IDF라는 값을 사용해 텍스트에서 특징을 추출하는 방법이다.                            |
| HashingVectorizer | Countvectorizer와 동일한 방법이지만, 해시 함수를 사용하기 때문에 실행시간에서 이득을 갖을 수 있다. |
